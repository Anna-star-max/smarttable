# SmartTable 智析平台

## 產品介紹

SmartTable 是一款 AI 驅動的內容智能解析工具，可自動識別內容類型並結構化輸出。

**支援內容類型：**
- 📋 會議記錄 → 行動項目表格 + 甘特圖
- 📊 數據表格 → 結構化表格
- 📝 合同法務 → 條款分析
- 📈 報告分析 → 摘要與建議
- 🎓 課程演講 → 大綱與知識點
- 💰 財務報表 → 數值統計

**核心功能：**
- 🎤 語音輸入（麦克风实时转文字）
- 📁 多格式檔案上傳（TXT / Word / 圖片OCR）
- 📈 甘特圖任務時間線
- 📊 統計面板（待處理/進行中/已完成）
- 📥 一鍵下載 Excel

## 快速開始

### 必要設定

1. **複製 `.env.example` 為 `.env`**，填入 MiniMax API Key
2. **雙擊執行 `安裝依賴.bat`**（首次使用或更換電腦時執行一次）
3. **雙擊執行 `一鍵啟動.bat`** 啟動服務

### 員工訪問

瀏覽器輸入：`http://伺服器IP:8000`
